# Reproducibility Checklist

Before using this repository for paper submission, replace any preview values in the
manuscript with results produced by this codebase.

Recommended reporting items:

- Dataset preprocessing rules and filtering thresholds.
- Train, validation, and test split protocol.
- Random seeds or fold identifiers.
- Hyperparameters used for SD-CDM and baselines.
- Mean and standard deviation over folds.
- Statistical tests against the strongest baseline.
- Hardware and software environment.
